✨Circuit Application✨

An application was developed in C# as a Year 1 project that uses logical principles to build and simulate circuits. The tool allows users to design circuits based on various logical operations, providing a hands-on approach to understanding circuit construction and behavior. The project focused on the following key aspects:

  • Circuit Design: Users can design circuits based on logical operations, helping them visualize and understand circuit behavior.
  • Logic Gates Implementation: Focused on implementing core concepts of logic gates and digital circuit design.
  • Simulation: Simulated the behavior of circuits, providing a practical way to experiment with logic and digital principles.

This project provided valuable experience in the following areas:

  • C# Programming: Gained hands-on experience in C# programming to implement the tool's functionality.
  • Problem-Solving: Applied problem-solving skills to design and simulate digital circuits.
  • Software Development: Gained experience in software development through the creation of a simulation tool for learning logic and circuits.

Overall, this project helped reinforce understanding of fundamental computing principles and provided a solid foundation for future software development projects.

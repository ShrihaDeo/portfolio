✨Once Upon A Time✨

An Interactive Authoring Environment was developed using React, JavaScript/TypeScript, HTML, and CSS. The project aimed to create an immersive platform for storytelling, where the backend was integrated with OpenAI to send requests to the ChatGPT API and display responses dynamically on the frontend.

My role in the project included the following:

  • Backend Integration: Ensured smooth communication between the frontend and ChatGPT for dynamic storytelling interactions.
  • Sound Integration: Developed a feature where a 3D stylus, when hovered over an image, would trigger sound playback and cause the pen to vibrate, enhancing user engagement and sensory interaction.

This project combined several key elements:

  • AI Integration: Utilized OpenAI to provide dynamic and interactive storytelling features.
  • User Experience Design: Focused on creating an engaging and intuitive interface for the users.
  • Multimedia Interaction: Incorporated sound and sensory features to increase immersion and interactivity.

This project provided valuable hands-on experience in both frontend and backend development, while exploring innovative features for interactive storytelling.
